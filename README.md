# SunShineMesh.github.io

The visual explainer for **MeshCredit** — the on-ledger trust bureau for AI agent payments on the XRP Ledger.

- 🔗 **Live site:** https://sunshinemesh.github.io
- 💻 **Code:** https://github.com/SunShineMesh/SunShine

A static, self-contained site (`index.html` + `assets/`) covering the idea, the cross-border payment
flow, the trust passport and KYA scorecard, the XRPL features/amendments used, and **real XRPL testnet
results** with live explorer links. The interactive live demo runs locally against testnet (see the code repo).

Built for SwissHacks 2026 · Ripple challenge — Know Your Agent (KYA).
